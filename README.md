<h2>Environment</h2>

>C lion 2019.1.4
>
>MinGW 5.0
>
>CMake 3.14.3
>
>Bundled GDB 8.2

일부 자료구조는 Visual Studio에서 테스트 해 위 환경에서 동작하지 않을 수 있음


<h2>Caution</h2>


- 프로젝트 경로에 한글명이 있으면 빌드되지 않음
- Run 버튼이 활성화되지 않는다면,  `Tools > CMake > Reload cmake Project` 를 클릭하면 활성화 된다.
